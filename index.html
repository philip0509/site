SKILLS

HTML5

Hypertext Markup Language (하이퍼 텍스트 마크업 언어)의 줄임말로 웹페이지의 기본적인 구조를 만들때 사용하는 언어입니다. HTML에서 제공하는 다양한 태그를 이용하여 웹페이지의 기초 골격을 만들어가는 언어라고 정의할 수 있습니다.

CSS

Cascading Style Sheets (캐스캐이딩 스타일 시트)의 줄임말로 웹페이지의 디자인, 스타일 부분을 담당하는 언어입니다. HTML로 웹페이지의 구조를 정의했다면, CSS로는 그 구조를 조금 더 보기 좋게 꾸며주는 언어라고 정의할 수 있습니다.

JAVASCRIPT

HTML이 기본적인 구조, CSS가 그 구조의 세부적인 디자인을 담당하는 언어였다면, JAVASCRIPT는 이전 두 언어로 만들어진 웹페이지의 동작 부분을 담당하는 언어입니다. 웹페이지에서 할 수 있는 동작 관련 기능을 JAVASCRIPT로 구현합니다.
